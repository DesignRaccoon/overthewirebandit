# Over The Wire Bandit

Deze repository documenteert mijn voortgang binnen de OverTheWire Bandit wargame als onderdeel van mijn omscholing van Software Developer naar Cybersecurity specialist.

Bandit richt zich op het leren van Linux fundamentals en command-line vaardigheden, die de basis vormen voor ethical hacking en penetration testing.


## Doel van deze repository

Het doel van deze repository is om mijn leerproces binnen cybersecurity inzichtelijk te maken door praktijkgerichte opdrachten uit te werken en te documenteren.

Per level leg ik vast:
- Welke uitdaging opgelost moest worden
- Welke stappen ik heb genomen
- Welke Linux-commando’s zijn gebruikt
- Wat ik hiervan heb geleerd

Op deze manier bouw ik niet alleen kennis op, maar ook een overzicht van mijn groei binnen cybersecurity.


##  Wat is OverTheWire Bandit?

OverTheWire Bandit is een cybersecurity wargame gericht op beginners en behandelt onder andere:

- Linux basisvaardigheden
- Werken via de command line
- Bestandsrechten en permissies
- SSH verbindingen
- Enumeratie technieken
- Probleemoplossend denken binnen systemen

Deze kennis vormt een belangrijke basis voor functies zoals:
- Ethical Hacker
- Penetration Tester
- Security Analyst
- Cybersecurity Engineer
- 

## Vaardigheden die ik oefen

Tijdens deze challenges werk ik aan:

- Navigeren binnen Linux systemen
- Bestands- en directorybeheer
- Werken met permissies
- Zoeken en analyseren van bestanden
- Remote toegang via SSH
- Security-minded probleemoplossing


##  Repository structuur

Elk Bandit level heeft een eigen map met daarin:

- Een volledige uitwerking
- Gebruikte commando’s
- Uitleg van de oplossing
- Screenshots van het proces
